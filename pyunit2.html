Lists

A list is a data structure in Python that is a mutable, or changeable, ordered sequence of elements. Each element or value inside a list is called an item. Lists allow you to group related data and are defined using square brackets [].

Creating Lists:

ages = [19, 26, 29]
print(ages)  # [19, 26, 29]

You can store elements of different data types in a list:

student = ['Jack', 32, 'Computer Science', [2, 4]]
print(student)

Accessing Elements:

languages = ['Python', 'Swift', 'C++']
print(languages[0])  # Python
print(languages[-1]) # C++ (last element)

Slicing:

my_list = ['p', 'r', 'o', 'g', 'r', 'a', 'm']
print(my_list[2:5])     # ['o', 'g', 'r']
print(my_list[0:3])     # ['p', 'r', 'o']

Modifying a List:

colors = ['Red', 'Black', 'Green']
colors[0] = 'Purple'
colors[2] = 'Blue'
print(colors)

Adding Elements:

fruits = ['apple', 'banana']
fruits.append('cherry')
fruits.insert(1, 'orange')
print(fruits)

Extending a List:

numbers = [1, 2, 3]
more_numbers = [4, 5, 6]
numbers.extend(more_numbers)
print(numbers)

Removing Elements:

numbers = [2, 4, 7, 9]
numbers.remove(4)
print(numbers)

List Methods: append(), insert(), extend(), remove(), pop(), clear(), index(), count(), sort(), reverse(), copy()

Iterating Through a List:

for fruit in fruits:
    print(fruit)

Dictionaries

A dictionary is an unordered collection of key-value pairs. Each key is unique and is used to access the corresponding value.

capitals = {'Germany': 'Berlin', 'Canada': 'Ottawa'}
print(capitals['Germany'])

Adding and Modifying:

capitals['Italy'] = 'Rome'
capitals['Germany'] = 'Munich'

Removing:

del capitals['Canada']

Iterating:

for country, city in capitals.items():
    print(country, city)

Sets

A set is an unordered collection of unique elements. It automatically removes duplicates and is ideal for membership testing and mathematical operations like union and intersection.

numbers = {1, 2, 3, 3, 2}
print(numbers)  # Output: {1, 2, 3}

Adding and Updating:

numbers.add(4)
numbers.update([5, 6])

Removing:

numbers.discard(2)

Tuples

A tuple is similar to a list but is immutable. Once created, its elements cannot be changed.

my_tuple = (1, 2, 3)
print(my_tuple[0])  # Output: 1

Immutability:

# my_tuple[0] = 4  # This will raise a TypeError

Length and Iteration:

for item in my_tuple:
    print(item)
