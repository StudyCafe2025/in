Rank of a Matrix by Echelon Form

To find the rank of a matrix using echelon form, reduce the matrix to row echelon form (or reduced row echelon form) and then count the number of non-zero rows; this count represents the rank of the matrix.

1. What is Echelon Form?

  • All non-zero rows are above any zero rows.
  • The first non-zero entry (pivot) of each row is to the right of the one in the row above.

In reduced row echelon form:

  • The leading entry is 1.
  • It is the only non-zero entry in its column.

2. How to Find the Rank:

  1. Reduce to (reduced) row echelon form using elementary row operations.
  2. Count non-zero rows. This is the matrix's rank.

Normal Form of a Matrix

The normal form (canonical form) is a simplified matrix using row and column operations where each row has at most one 1 and others are 0.

Key Characteristics:

  • Only zeros and ones.
  • One non-zero (1) per row max.
  • Pivots are the 1s used to determine rank.

Solving Linear Equations

To solve homogeneous and non-homogeneous systems, you can use row reduction or matrix methods.

Homogeneous (Ax = 0)

  • Always has the trivial solution (x = 0).
  • If determinant = 0 → infinite solutions.

Non-homogeneous (Ax = b)

  • Consistent if rank(A) = rank(A|b).
  • Unique if rank = number of unknowns.
  • Infinite if rank < number of unknowns.

Eigenvalues & Eigenvectors

These describe how a linear transformation scales vectors.

Definitions:

  • Eigenvalue: Scalar λ such that Av = λv.
  • Eigenvector: Non-zero vector v that remains on the same line after transformation.

Key Properties:

  • Only square matrices have eigenvalues/vectors.
  • Eigenvectors ≠ 0; eigenvalues can be zero.
  • Sum = trace of matrix, product = determinant.
