Cayley-Hamilton Theorem

The Cayley-Hamilton theorem states that every square matrix satisfies its own characteristic equation.

If you substitute the matrix into its characteristic polynomial, the result is the zero matrix.

Characteristic Polynomial:

For a square matrix A, the characteristic polynomial is: p(x) = det(xI - A)

Theorem Statement:

If p(x) is the characteristic polynomial of A, then p(A) = 0.

Example (2x2 Matrix):

If A = [[a, b], [c, d]], then p(x) = x² - (a+d)x + (ad - bc), and substituting A gives:

A² - (a+d)A + (ad-bc)I = 0

Using C-H to Find Inverse and Powers

  1. Find the characteristic polynomial: det(λI - A)
  2. Apply C-H: p(A) = 0
  3. Solve for A⁻¹ using: A³ + aA² + bA + cI = 0 → isolate A⁻¹

Higher powers like A⁴ can be computed as combinations of A², A, and I.

Example:

Given p(λ) = λ² - 5λ + 6, then:

A² - 5A + 6I = 0 → A⁻¹ = (-A + 5I)/6

Diagonalization of a Matrix

Diagonalization transforms a matrix A into a diagonal matrix D such that:

A = PDP⁻¹, where P is a matrix of eigenvectors.

  1. Find eigenvalues and eigenvectors
  2. Construct P and D
  3. Verify A = PDP⁻¹

Used for simplifying matrix powers: Aⁿ = PDⁿP⁻¹

Quadratic Forms & Their Nature

A quadratic form is of the type: Q(x) = xᵀAx

It is classified based on its eigenvalues:

  • Positive Definite: All eigenvalues positive
  • Negative Definite: All negative
  • Indefinite: Mixed signs

Reducing to Canonical Form via Orthogonal Transformation

Steps:

  1. Represent Q(x) as xᵀAx with symmetric A
  2. Find eigenvalues/eigenvectors
  3. Form orthogonal matrix P
  4. Transform: Q(x) = yᵀDy where D is diagonal

This eliminates cross terms and yields canonical form: λ₁y₁² + λ₂y₂² + ...
