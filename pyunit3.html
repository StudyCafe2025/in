What is a Function?

A function is a named block of code designed to perform a specific task when it is called. Functions help in reducing repetition, increasing modularity, and making the code more organized and easier to debug.

Program Routines

Program routines are individual blocks of code (like functions or procedures) that perform specific tasks within a program. These routines are the building blocks of structured programming.

def welcome():
    print("Welcome to Python Programming!")

welcome()

Types of Functions

  • Built-in Functions: print(), len(), type()
  • User-defined Functions: created using def

Function with Parameters

def greet(name):
    print("Hello", name)

greet("Alice")  # Output: Hello Alice

Return Statement

def square(n):
    return n * n

print(square(6))  # Output: 36

Default Parameters

def greet(name="Guest"):
    print("Welcome", name)

greet()  # Output: Welcome Guest

Keyword Arguments

def profile(name, age):
    print(f"{name} is {age} years old")

profile(age=21, name="Sara")  # Output: Sara is 21 years old

Variable-Length Arguments

def total(*nums):
    return sum(nums)

print(total(1, 2, 3, 4))  # Output: 10

Scope of Variables

Local Scope: Defined within a function.

Global Scope: Defined outside all functions.

x = 10  # Global

def show():
    x = 5  # Local
    print(x)

show()       # Output: 5
print(x)     # Output: 10

Modules

A module is a Python file (.py) containing code (functions, variables, or classes) that can be imported and reused in other programs.

Creating a Custom Module

# calculator.py
def add(a, b):
    return a + b

Using the Module

import calculator

print(calculator.add(5, 3))  # Output: 8

Python Built-in Modules

  • math – Mathematical functions
  • random – Generate random numbers
  • datetime – Date and time operations
  • os – Operating system tasks
import math
print(math.pi)              # Output: 3.141592653589793
print(math.factorial(5))    # Output: 120

Top-Down Design

Top-down design is a problem-solving strategy where you break a large task (main function) into smaller sub-tasks (sub-functions), making development easier and more organized.

def calculator():
    a = int(input("Enter number 1: "))
    b = int(input("Enter number 2: "))
    print("Sum:", add(a, b))

def add(x, y):
    return x + y

calculator()

Importing Modules

Importing Specific Functions

from math import sqrt
print(sqrt(16))  # Output: 4.0

Using Aliases

import math as m
print(m.pow(2, 3))  # Output: 8.0

Custom Module Example

# myutils.py
def welcome():
    print("Welcome to custom module!")

# main.py
import myutils
myutils.welcome()
