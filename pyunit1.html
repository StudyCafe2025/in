Introduction to Python

Python is a high-level, interpreted, and dynamically typed programming language known for its simplicity and readability. It's used across various domains such as web development, data science, automation, and more. It emphasizes code readability and allows developers to write fewer lines of code to achieve the same task compared to other languages.

What is a Computational Problem?

A computational problem is one that a computer can solve. It involves determining a solution using algorithms under resource constraints like time and memory. These problems are fundamental to theoretical computer science.

Process of Computational Problem Solving

  • Analyzing the Problem: Understanding the input, output, and constraints.
  • Developing an Algorithm: Creating a sequence of steps to solve the problem.
  • Coding: Translating the algorithm into Python code.
  • Testing and Debugging: Running test cases to validate and fix the code.

History of Python

Developed by Guido van Rossum at CWI in the Netherlands and released in 1991. Python 1.0 introduced lambda, map, filter. Python 2.0 added garbage collection and list comprehensions. Python 3.0 was a major revision. The name "Python" was inspired by the TV show “Monty Python’s Flying Circus.”

What is Python?

Python is a general-purpose, high-level, dynamically typed, interpreted, and object-oriented programming language. It supports multiple programming paradigms and promotes code readability.

Features of Python

  • Simple and Easy to Learn
  • Free and Open Source
  • Platform Independent
  • Dynamically Typed
  • Both Procedure and Object-Oriented
  • Extensible and Embeddable
  • Extensive Library Support

Advantages of Python

  • Open-source and free
  • Beginner-friendly syntax
  • Third-party module support
  • Platform-independent
  • Great online community

Limitations of Python

  • Not ideal for mobile apps
  • Slow execution due to interpretation
  • High memory consumption
  • Less efficient database connectivity

Literals

Fixed values assigned directly to variables.

Numeric:

a = 7
b = 8.3
c = -15
d = 2+3j

String:

a = 'Hello'
b = "Krishna"
c = '''Python is good for learning'''

Boolean:

a = 3
b = (a == 3)
c = True + 10

Special:

var = None

Variables and Identifiers

Variables are named memory locations. Identifiers are names given to program elements.

  • Start with a letter or underscore
  • No special characters
  • Cannot be keywords
  • Case-sensitive

Python Keywords

Reserved words like if, else, for, while, def, etc. cannot be used as identifiers.

Operators

  • Arithmetic: +, -, *, /, //, %, **
  • Comparison: ==, !=, >, <, >=, <=
  • Assignment: =, +=, -=, *=, /=, %=, //=, **=
  • Logical: and, or, not
  • Identity: is, is not
  • Membership: in, not in
  • Bitwise: &, |, ^, ~, <<, >>

Expressions

Combinations of literals, variables, and operators that evaluate to a value.

5 + 3
x * 2
name == "Alice"
len("hello")

Data Types

  • Numeric: int, float, complex
  • Sequence: str, list, tuple, range
  • Set: set, frozenset
  • Mapping: dict
  • Boolean: bool
  • Binary: bytes, bytearray, memoryview
  • None: NoneType

Indentation in Loops

Python uses indentation to define blocks of code.

for lang in ['Swift', 'Python', 'Go']:
    print(lang)
    print('-----')
print('Last statement')

Control Structures

Boolean Expressions:

Evaluate to True or False. Used in conditional and loop statements.

Selection Control:

if condition:
    # code block
elif condition:
    # code block
else:
    # code block

Iterative Control:

for lang in ['Swift', 'Python']:
    print(lang)

while num != 0:
    print(num)
    num = int(input("Enter a number: "))

Loop Control Statements

  • break: Exits the loop immediately
  • continue: Skips to the next iteration
  • pass: Placeholder that does nothing
for lang in ['Swift', 'Python', 'Go']:
    if lang == 'Go':
        break
    print(lang)
