1. Text File

Definition :A text file is a file that contains readable characters (like letters, numbers, and symbols). These are saved with a .txt extension and are used for storing data in plain text format.

➤ Description:
A text file is a simple file that contains text without any special formatting (like bold, italic, etc.). The contents can be viewed and edited using any text editor. Python treats text files as sequences of characters.

➤ Characteristics:
Usually have extensions like .txt, .csv, .log
Store data in plain, human-readable format
Useful for storing logs, configuration settings, and exported data

➤ Example:

# Create and write to a text file
file = open("sample.txt", "w")
file.write("Welcome to Python file handling!")
file.close()

Example :

# Writing to a text file
with open("example.txt", "w") as file:
    file.write("Hello, world!")

2. Using Text Files

You can perform operations like reading, writing, and appending using built-in functions in Python. Python provides built-in functions to interact with files: open(), read(), write(), close()

➤ File Modes:

Mode Description
'r' Read (default)
'w' Write (overwrite)
'a' Append
'r+' Read and Write
'b' Binary mode (e.g. 'rb')

a. Reading a file:

with open("example.txt", "r") as file:
    content = file.read()
    print(content)

➤ Examples:

Reading from a file:
with open("sample.txt", "r") as f:
    print(f.read())  # Reads entire content
Writing to a file:
with open("sample.txt", "w") as f:
    f.write("Overwriting the file content.")
Appending to a file:
with open("sample.txt", "a") as f:
    f.write("\nAppended text.")
Reading line by line:
with open("sample.txt", "r") as f:
    for line in f:
        print(line.strip())  # removes \n

b. Appending to a file:

with open("example.txt", "a") as file:
    file.write("\nThis is a new line.")

c. Writing to a file (overwrites content):

with open("example.txt", "w") as file:
    file.write("New content")

3. String Processing

Involves manipulating strings read from or written to text files. Once you read content from a file, it is stored as a string. You can use string methods to manipulate it. Useful for data cleaning or formatting.

➤ Common String Methods:

  • .strip() – removes whitespace/newlines
  • .lower() / .upper() – changes case
  • .split() – splits a string into a list
  • .replace(old, new) – replaces substrings

Example :

with open("example.txt", "r") as file:
    for line in file:
        line = line.strip()          # Removes extra spaces and newline characters
        words = line.split()         # Splits line into list of words
        print(words)

➤ Example:

with open("data.txt", "r") as f:
    for line in f:
        clean = line.strip()
        words = clean.split()
        print("Words:", words)

➤ Real-world use:

Reading and analyzing logs, filtering data, formatting reports, etc.

4. Exception Handling

Used to handle errors that may occur while working with files (e.g., file not found, permission denied). Exception handling ensures your program doesn’t crash if something goes wrong during file operations.

➤ Common File Exceptions:

  • FileNotFoundError: File doesn’t exist
  • PermissionError: No permission to read/write
  • IOError: General input/output error

Example :

try:
    with open("nonexistent.txt", "r") as file:
        content = file.read()
except FileNotFoundError:
    print("The file was not found.")
except IOError:
    print("An error occurred while reading the file.")

➤ Example:

try:
    with open("missing.txt", "r") as f:
        content = f.read()
except FileNotFoundError:
    print("File not found. Please check the name.")
except IOError:
    print("An I/O error occurred.")
finally:
    print("Execution completed.")

➤ Why it's important:

Prevents abrupt program termination
Helps in debugging
Improves user experience
