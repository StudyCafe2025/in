Taylor's and Maclaurin's Theorems

Taylor's Theorem approximates functions near a point a using derivatives. Maclaurin's Theorem is the special case where a = 0.

Formula:

Pn(x) = f(a) + f'(a)(x-a) + f''(a)(x-a)2/2! + ... + f(n)(a)(x-a)n/n!

Maclaurin:

Same formula with a = 0. Used for functions like sin(x), ex, etc.

Example:

Maclaurin for sin(x): x - x3/6

Partial Derivatives

Measures rate of change of multivariable functions with respect to one variable, keeping others constant.

Example:

f(x, y) = x2 + 2xy + y2

∂f/∂x = 2x + 2y, ∂f/∂y = 2x + 2y

Total Derivatives

Captures how a multivariable function changes when all variables change simultaneously.

Formula:

dz/dt = (∂z/∂x)(dx/dt) + (∂z/∂y)(dy/dt)

Maxima and Minima

To find critical points: Set partial derivatives to zero, then apply second derivative test using D = fxxfyy - (fxy)2

Lagrange Multipliers

Used to find local maxima/minima of functions with constraints. Solve the system: ∇f = λ∇g and g(x, y, z) = 0

Example:

Minimize f(x, y, z) = x2 + y2 + z2 given x + y + z = 1
